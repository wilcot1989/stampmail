# NeatStamp Product Roadmap

## Fase 1: Table Stakes (Week 1-2)
*Zonder dit geen serieuze concurrent voor teams*

### Chrome Extension
- 1-click signature install in Gmail
- Detecteert signature settings, plakt HTML
- "Powered by NeatStamp" in Chrome Web Store
- Gratis distributie, viraliteit

### Google Workspace Marketplace App
- Admin deployt signatures naar alle medewerkers
- Vereist voor team sales
- Integratie met Google Directory

### Master Signature Template
- 1 template aanmaken, medewerker data invullen via CSV
- Of automatisch via directory sync
- Wijziging in template → update overal

---

## Fase 2: Wow Factor (Week 3-4)
*Dingen die NIEMAND heeft*

### AI Signature Optimizer
- Upload/plak je huidige signature
- AI analyseert en geeft score 0-100
- Concrete verbeterpunten: "Missing phone number (+22% response rate)"
- "Fix it with NeatStamp" CTA → conversie
- Viral: "What's your email signature score?" op LinkedIn

### Signature Score
- Automatische score per signature
- Criteria: foto, social links, CTA, lengte, Outlook-safe, mobile responsive
- Gamification: "Your score: 72/100 — add a headshot to reach 85"
- Dashboard widget

### Live Email Client Preview
- Toon signature side-by-side in: Gmail, Outlook Desktop, Outlook Mobile, Apple Mail
- Niemand doet dit — lost het #1 probleem op
- Trust builder: "We TONEN je dat het werkt"

### Smart Banner Scheduling
- "Toon banner A van 1-15 maart, dan switch naar banner B"
- Per afdeling, per campagne
- Beter dan Newoldstamp's basic versie

## Fase 3: Enterprise Killer (Maand 2-3)
*Stelen van WiseStamp/Exclaimer klanten*

### Microsoft 365 Integration
- Auto-deploy via Exchange transport rules
- Outlook add-in
- Server-side signature injection

### Directory Sync
- Active Directory / Azure AD
- Google Workspace Directory
- Auto-update bij wijzigingen (titel, telefoon, afdeling)

### A/B Testing
- Test 2 signatures tegen elkaar
- Meet opens, clicks, response rates
- Automatisch de winnaar selecteren

### CRM Integratie
- HubSpot: signature clicks verschijnen als activiteit
- Salesforce: signature engagement in contact record
- PipeDrive: link clicks als touchpoint

### Agency Dashboard
- Beheer signatures voor meerdere klanten
- White-label optie
- Per-client billing

## Fase 4: Moonshot (Maand 3+)
*De markt versteld doen staan*

### Signature API
- REST API voor andere apps
- Genereer signatures programmatisch
- €0.01 per signature call
- Documentatie + SDK

### Dynamic Signatures
- Signature verandert per ontvanger
- Sales lead → toon case study
- Bestaande klant → toon support link
- Prospect → toon pricing CTA

### CRM Analytics
- "Deze prospect klikte 3x op je pricing link"
- Real-time notificaties
- Lead scoring based on signature engagement

### Drag & Drop Editor
- Visuele editor met slepen en neerzetten
- Elementen vrij positioneren
- Real-time preview
- Export als Outlook-safe HTML
